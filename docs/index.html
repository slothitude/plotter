The Scriptorium

Single-stroke cursive font with automatic letter chaining. Lowercase letters flow into continuous pen strokes — no lift, no gaps. Also includes classic Hershey vector fonts.

Script Simplex Cursive — connected lowercase strokes rendered as continuous pen paths. The quick brown fox jumps over the lazy dog.
Scriptorium Panel — font selection, size, word-wrap, and instant canvas preview

5-Step Workflow

From connection to finished print in five focused steps. Each step has its own canvas preview and dedicated tools.

Setup — serial connection, calibration, page size, bed leveling
Create — upload SVG, generate patterns, trace images, write text
Test Patterns — circles, grids, spirals, stars, calibration marks
Prepare — transforms, preview, convert to G-code
Plot — print with live progress, stats, speed control
Config — tool profiles, movement settings, water control

Manga Toolkit

Generate manga-specific plotter art: panels, speed lines, dot tone, speech bubbles, SFX, and full page compositions.

Manga Panel — panel layouts, speed lines, tone fills, speech bubbles

Feature Overview

SVG to G-code

Full pipeline: parse SVG, flatten curves, RDP simplification, path optimization, pen offset compensation, auto-centering. One click.

Live Plot

Draw on Wacom Slate via BLE, see strokes plotted in real-time. Portrait A5 mapping with coordinate rotation.

Watercolor 2-Pass

Pass 1: dry pencil guide layer. M0 pause for tool swap. Pass 2: wet brush retraces with periodic water cup dips.

Cursive Font

Script Simplex with automatic letter chaining. Lowercase letters connect into continuous strokes. Word-wrap with paragraph support.

Tool Profiles

Pencil, pen, watercolor — each with independent speeds, Z heights, XY offsets. Calibration persisted in JSON, overriding TOML defaults.

Handwriting OCR

Capture ink from Wacom Slate, auto-crop, detect text regions, transcribe via vision LLM chain (NVIDIA NIM → ZhipuAI → Ollama).
